Class: Compose

Compose

Constructor

new Compose(lensA, lensB) → {Compose}

Create the composite of two Lenses. For instance, to create a Lens for arr[i][j]:

arrIJLens = new Compose(
    new IndexedLens(1),
    new IndexedLens(0)
);

Then arrIJLens.get, .set, and .over first call the appropriate function for the first Lens, then the appropriate function for the second Lens and return the result.
